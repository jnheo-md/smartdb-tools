Powerful query capabilities

Schema Exploration

Browse hospitals, tables, and variables. Search across 3,000+ clinical data points with fuzzy matching.

ysr3 schema search YSU "blood pressure" ysr3 schema variable YSU NIHSS_total

Patient Queries

Query patient data with automatic table JOINs. Filter by any variable with support for complex conditions.

ysr3 query data YSU \ --vars "pt_sex,pt_age,NIHSS_total" \ --filters '[{"variable":"Thr_mechanical","operator":"=","value":"1"}]'

Outcome Analysis

Get mRS outcomes from the cohort table with automatic death imputation. The only correct way to analyze stroke outcomes.

ysr3 query followup YSU --period 3m ysr3 export followup YSU -p 3m -v "pt_sex,pt_age"

Clot Composition

Access thrombus histology data from the ARIA study — platelet, fibrin, RBC, and neutrophil composition of retrieved clots.

ysr3 clot patients YSU ysr3 clot composition YSU YSU001

Built with security in mind

No Direct DB Access

All queries go through an authenticated HTTPS API. No database credentials ever touch your machine.

Role-Based Access

Server-side permission enforcement. Data access is limited to your authorized scope.

Secure Sessions

JWT tokens stored with 0600 permissions, 24-hour automatic expiry, atomic file writes.
